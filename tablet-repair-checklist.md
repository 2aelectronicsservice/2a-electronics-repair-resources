# Tablet Repair Checklist

This checklist helps inspect common tablet problems before repair.

## Screen and Touch

- Check for cracked glass
- Test touch response across the full screen
- Look for dead spots, ghost touch, or display lines
- Test brightness and color quality

## Charging and Battery

- Inspect the charging port
- Test with a known-good charger and cable
- Check if the tablet charges normally
- Check for fast battery drain or overheating

## Buttons and Cameras

- Test power button
- Test volume buttons
- Test front and rear cameras
- Test speaker and microphone

## Connectivity

- Test Wi-Fi
- Test Bluetooth
- Check cellular connection if the tablet supports SIM service
- Test charging and data connection to a computer

## System Check

- Confirm the tablet powers on
- Check for passcode or account lock
- Check for software update problems
- Test basic apps and settings

## Final Notes

This checklist is for general inspection. Tablet repairs should be done carefully using proper tools and parts.
