# Game Console Repair Checklist

This checklist helps inspect common game console problems before repair.

## Power

- Check if the console powers on
- Test the original power cable
- Check for random shutdowns
- Check for overheating or loud fan noise

## HDMI and Display

- Inspect the HDMI port for damage
- Test with a known-good HDMI cable
- Test with another TV or monitor
- Check for no signal, flickering, or display cutouts

## Disc Drive

- Test if the disc drive accepts discs
- Check if discs spin normally
- Listen for grinding or clicking sounds
- Test if games or movies are readable

## Storage and System

- Check if the console boots to the home screen
- Test storage errors or system update errors
- Check for slow loading or freezing
- Reinstall system software if needed

## Controller and Connectivity

- Test controller pairing
- Test USB ports
- Test Wi-Fi and Ethernet
- Test Bluetooth if supported

## Cleaning and Maintenance

- Check for dust buildup
- Inspect vents and fan area
- Replace thermal paste when needed
- Clean carefully using proper tools

## Final Notes

This checklist is for general inspection. Console repairs should be done carefully to avoid board damage.
