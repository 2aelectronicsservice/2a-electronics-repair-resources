# Phone Repair Diagnostic Checklist

This checklist helps identify common phone problems before repair.

## Screen and Touch

- Check if the screen turns on
- Test touch response across the full display
- Look for dead spots, ghost touch, lines, or black spots
- Check brightness and color quality

## Charging

- Inspect the charging port for dust, damage, or looseness
- Test with a known-good charging cable
- Check if the phone charges normally
- Check if the phone connects to a computer

## Battery

- Check if the phone powers on
- Check if the battery drains fast
- Check if the device restarts by itself
- Check for swelling or overheating

## Cameras and Audio

- Test front and rear cameras
- Test speaker and microphone
- Test calls, recording, and video sound

## Connectivity

- Test Wi-Fi
- Test Bluetooth
- Test cellular signal
- Test SIM card recognition

## Final Notes

This checklist is for general inspection. Some repairs should be done by experienced technicians using proper tools.
