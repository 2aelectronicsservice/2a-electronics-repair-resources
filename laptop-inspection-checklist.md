# Laptop Inspection Checklist

This checklist helps inspect used laptops before repair, resale, or buyback.

## Power and Charging

- Check if the laptop powers on
- Test the charger and charging port
- Check if the battery charges
- Check if the laptop shuts off randomly
- Check for overheating or fan noise

## Screen and Display

- Check for cracks, lines, black spots, or flickering
- Test brightness control
- Test external display output if needed
- Check hinges and screen movement

## Keyboard and Touchpad

- Test all keyboard keys
- Test the touchpad and click buttons
- Check for liquid damage signs
- Check keyboard backlight if available

## Ports and Connectivity

- Test USB ports
- Test HDMI or video output
- Test headphone jack
- Test Wi-Fi and Bluetooth
- Check webcam and microphone

## Storage and System

- Check SSD or HDD health
- Confirm the operating system boots
- Check for password locks or account locks
- Confirm basic device specifications

## Final Notes

This checklist is for general laptop inspection. Some problems require deeper diagnostics by an experienced technician.
